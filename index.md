## 🔍 Frequently asked questions

### Will this software change my original file?
The editor creates a copy of your document by default. This keeps your original file safe from accidental changes. You can save your work as a new file, or overwrite the old one if you prefer to save space.

### Can I run this on a Mac?
This specific version works only on Windows. It does not support macOS or Linux systems at this time.

### Does the OCR tool work with handwriting?
The OCR tool performs best with clear, printed text. Handwriting often results in errors because the software struggles to map scripts to character profiles. Use the tool primarily for typed documents and scanned invoices.

### How do I update to the latest version?
Check the download page periodically for new releases. When a new version arrives, download the installer and run it over the old one. The installer handles the update process automatically and keeps your settings intact.

### Is my data secure?
All data stays on your local machine. The internal processing of your documents happens within your computer hardware. No external server parses your files during the standard edit or compress operations.
